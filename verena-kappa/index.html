Importe os CSVs da Duda e do Luka para calcular o Kappa

Como usar este aplicativo
1. Importe o CSV das avaliações da Duda.
2. Importe o CSV das avaliações do Luka.
3. O sistema junta os dados automaticamente.
4. Clique em Calcular Kappa.
5. O app compara apenas cenários com o mesmo scenario_id avaliados por ambos.
